# 3dtiles_cube

Example code to create a cube at Dam square, a minimal example to create 3D Tiles

Create input table 

```sql
CREATE TABLE cube (
    id SERIAL PRIMARY KEY,
    name VARCHAR(255),
    geom GEOMETRY(GeometryZ, 4326)
);
```

Insert the cube

Run pg2b3dm

```
pg2b3dm -U postgres -h localhost -p 5432 -d postgres -t cube -c geom -a name
```
